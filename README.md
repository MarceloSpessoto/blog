# kw blog

This is the Jekyll files for the kw blog page. See:

# Do you want to write a blog post for kw page?

It is your lucky day since this is super simple; just follow the following
steps:

1. Fork this repository
2. Create a blog post file at _posts/ (make sure you follow the file pattern)
3. Write
4. Send your PR

That's it.
